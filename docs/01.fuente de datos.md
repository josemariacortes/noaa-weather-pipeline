# 01. Fuentes de Datos

## Origen de los datos

**Nombre del dataset**: NOAA Global Historical Climatology Network - Daily (GHCN-Daily)  
**Proveedor**: National Centers for Environmental Information (NCEI) – NOAA  
**Tipo de datos**: Observaciones meteorológicas diarias a nivel global  
**Formato original**: Archivos CSV comprimidos en gzip (un archivo por año)  
**Fuente oficial**: [https://www.ncei.noaa.gov/pub/data/ghcn/daily/by_year/](https://www.ncei.noaa.gov/pub/data/ghcn/daily/by_year/)

### Características principales

- Datos diarios desde 1763 hasta la actualidad.
- Más de 100.000 estaciones meteorológicas en todo el mundo.
- Actualización diaria.
- Licencia: Dominio público (datos abiertos del gobierno de EE.UU.).

### Uso en este proyecto

Se utiliza el fichero local `data/{year}_.csv` como fuente para la capa **Bronze** del Lakehouse.  
El pipeline `dlt` lee estos ficheros y los guarda en formato Parquet particionado por año.
